四、数据收集与隐私说明

为向你提供服务，本站收集和使用以下数据：

4.1 账号信息

  • GitHub OAuth 登录：获取你的 GitHub 用户名、用户 ID 及公开邮箱信息，仅用于账户识别与违规追溯
  • 邮箱登录/注册：收集你的邮箱地址及密码（密码经 PBKDF2-SHA256 哈希加密存储，本站无法获取明文密码）

4.2 使用数据

  • IP 地址：你的所有操作（登录、注册、生图等）的 IP 地址均被记录，用于违规追溯、限流及安全防护
  • 生图内容：你提交的提示词（prompt）及生成的图片将被记录，仅用于审核及违规追溯

4.3 Cookie 使用说明

本站使用以下 Cookie：

  • session — 用户登录会话令牌，用于识别你的登录身份。有效期 30 天。功能性必要 Cookie
  • oauth_state — GitHub OAuth 登录时的 CSRF 安全防护。有效期 10 分钟。功能性必要 Cookie
  • genNoticeAcked — 记录你已确认生图公告，避免每次弹窗。有效期 1 年

4.4 第三方服务

  • GitHub OAuth：你使用 GitHub 登录时，你的数据将按 GitHub 隐私政策 处理
  • Cloudflare Turnstile：本站使用 Cloudflare Turnstile 进行人机验证。Cloudflare 可能收集你的 IP 地址及设备信息用于防滥用检测，详见 Cloudflare 隐私政策
  • Tailwind CSS CDN：本站前端使用 Tailwind CSS CDN 提供样式支持，CDN 服务商可能获取你的 IP 地址及浏览器信息

4.5 数据保留与删除

  • 生图日志：最多保留 20,000 条，超出后自动覆盖最旧的记录
  • 用户作品：你可以在「我的作品」中删除自己生成的图片，对应记录同步清除
  • 账号删除：如需删除账号及相关数据，请通过联系邮箱向我们提出请求

4.6 图片可见性

本站所有生成内容强制仅生图者本人可见。运营者为履行审核义务保留后台查看权限，但不会主动公开任何用户作品。仅在你授权的情况下，作品可能被列为精选展示。
